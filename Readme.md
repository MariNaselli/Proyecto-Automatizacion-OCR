# Transcriptor Automático Bilingüe (OCR)

Este proyecto es una herramienta web capaz de leer imágenes y convertirlas en texto editable de forma automática. 

### Características:
* **Detección Dual:** Reconoce texto en español e inglés simultáneamente.
* **Interfaz Minimalista:** Diseñada con Streamlit para una experiencia de usuario rápida y directa.
* **Descarga Instantánea:** Permite exportar el texto detectado a un archivo `.txt` con un solo clic.

### Tecnologías utilizadas:
* **Python**: Lenguaje principal.
* **Tesseract OCR**: Motor de reconocimiento de caracteres.
* **Streamlit**: Framework para la interfaz web.
* **Pillow**: Procesamiento de imágenes.
